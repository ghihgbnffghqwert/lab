Lab — Документація коду

Згенеровано за PHPDoc-коментарями. Для повної генерації виконайте: composer install && composer run-script docs.

Файли та елементи

  • index.php — головна сторінка (лендінг)
  • db.php — підключення до БД (глобальна змінна $conn)
  • lib/helpers.php — функція h(string $s): string — екранування HTML
  • actions/contact.php — обробник POST форми контактів

Функції

  • h($s) — екранує HTML-спецсимволи для безпечного виводу (захист від XSS).

Повна HTML-документація генерується phpDocumentor при виконанні composer run-script docs.
