# Архітектура проєкту

## Огляд

Сайт — односторінковий лендінг (SPL) на PHP без фреймворку. Рендеринг на сервері (PHP), стилі — Bootstrap 5, невелика клієнтська логіка (чат) у `public/chat.js`.

## Компоненти та взаємодія

```
[Користувач] → index.php → db.php (MySQLi) → MySQL
                    ↓
              lib/helpers.php (h())
                    ↓
              HTML (секції: hero, about, services, news, contact, chat)
                    ↓
[Форма контактів] → actions/contact.php → db.php → INSERT contact_requests
```

- **index.php** — єдина точка входу для головної сторінки: підключає БД, хелпери, виконує запити до `settings`, `services`, `news`, виводить HTML.
- **db.php** — створює одне глобальне з'єднання `$conn` (MySQLi), кодування UTF-8.
- **lib/helpers.php** — функція `h()` для екранування виводу (захист від XSS).
- **actions/contact.php** — обробка POST форми контактів: валідація, збереження в БД, редірект.

## Бізнес-логіка

- Налаштування сайту (назва, герой, про компанію, зображення) зберігаються в одному рядку таблиці `settings`.
- Послуги та новини — списки з таблиць `services` та `news`; зображення — у `uploads/services/`, `uploads/news/`.
- Контактна форма зберігає заявки в `contact_requests` (name, email, message, created_at). Листи по пошті в цій реалізації не відправляються.
- Чат — лише клієнтська демонстрація (повідомлення додаються в DOM без збереження на сервері).
