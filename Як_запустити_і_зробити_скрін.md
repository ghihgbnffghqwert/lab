# Як запустити проєкт і зробити скріншот

## 1. Запуск сайту (на вашому комп’ютері)

Відкрийте **Термінал** (на Mac — Термінал або iTerm) і виконайте:

```bash
cd "/Users/kostiantyn/Desktop/заказ"
php -S localhost:8765
```

Має з’явитися рядок на кшталт: `Development Server (http://localhost:8765) started`.

**Якщо команди `php` немає:**  
- Mac: встановіть PHP (`brew install php`) або запускайте через MAMP/XAMPP (скопіюйте проєкт у `htdocs` і відкривайте через їхній localhost).  
- Windows: встановіть PHP або використовуйте Open Server / XAMPP.

## 2. Відкрити в браузері

У браузері відкрийте:

- **Головна сторінка:** http://localhost:8765/
- **Документація (phpDoc):** http://localhost:8765/docs/phpdoc/
- **Swagger UI (API):** http://localhost:8765/docs/swagger.html

Сайт працює в **демо-режимі** без бази даних: показуються приклади послуг і новин.

## 3. Зробити скріншот

- **Mac:** `Cmd + Shift + 4` — виділіть область екрана; або `Cmd + Shift + 5` — виберіть «Знімок екрана» / «Вікно».
- **Windows:** `Win + Shift + S` (фрагмент екрана) або кнопка «Print Screen».

Знімок збережеться на робочому столі або в буфері обміну — вставте його в звіт або документ.

## 4. Зупинити сервер

У тому ж вікні терміналу натисніть **Ctrl + C**.
